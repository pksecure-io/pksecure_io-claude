### Limitations & Disclaimers

#### Data Accuracy

This investigation used publicly available sources and free OSINT methods. Important considerations:

- **Area Code Location**: Area codes indicate where numbers were originally assigned, not current user location. Mobile and VoIP numbers can be used anywhere.
- **Carrier Information**: Carrier data from free sources may be outdated due to number portability (users switching carriers while keeping their number).
- **Phone Number Reassignment**: Phone numbers are recycled and reassigned to new users. Historical data may refer to a previous owner if the number was reassigned.
- **Caller ID Spoofing**: Caller ID can be easily spoofed (faked). The number displayed on caller ID may not be the actual caller. This is extremely common in scam calls.
- **Spam Database Subjectivity**: Spam reports are user-submitted and may include false positives (legitimate businesses reported as spam). Always read user comments for context.
- **VoIP Detection Limitations**: Without paid APIs, VoIP detection is approximate and may be inaccurate.
- **Free Source Limitations**: Free sources cannot provide real-time carrier validation, live number status, or comprehensive social media searches.

#### Privacy & Legal Considerations

This report is intended for legitimate investigative, security, or research purposes only:

- **GDPR/CCPA Compliance**: Always respect privacy laws when conducting investigations. This report uses only publicly available information.
- **Platform Terms of Service**: Social media checks respect platform terms of service and privacy settings. No unauthorized scraping or API misuse was performed.
- **Ethical Use**: This tool should not be used for harassment, stalking, unauthorized surveillance, or any illegal purpose.
- **Cross-Verification Required**: Always cross-verify findings from multiple sources before taking action based on this report.
- **Not Legal Advice**: This report is for informational purposes only and does not constitute legal advice.

#### Technical Limitations

What this investigation **could not** determine with free sources:

- ❌ Real-time carrier information (requires paid API)
- ❌ Whether the number is currently active/in-service (requires HLR lookup)
- ❌ Current user's physical location (impossible without access to carrier data)
- ❌ Call history or CDRs (Call Detail Records) - private carrier data
- ❌ Comprehensive social media account linkage (privacy-protected)
- ❌ Definitive VoIP vs. landline classification (approximate only)

#### Recommended Actions

Based on the findings:

- **Cross-Reference**: Verify findings with multiple independent sources before taking action
- **Context Matters**: Consider the full context (time, frequency, content of calls/messages, caller behavior)
- **Paid Services**: For critical investigations, consider using paid verification services for higher accuracy
- **Report Scams**: If this appears to be a scam number, report it to:
  - FTC (Federal Trade Commission): https://reportfraud.ftc.gov/
  - FCC (Federal Communications Commission): https://www.fcc.gov/complaints
  - Local law enforcement (if threats or significant fraud)
- **Do Not Engage**: If identified as a scam, do not call back or engage with the caller

## Notes for Report Generation

When generating reports using this template:

1. **Replace all [bracketed placeholders]** with actual findings
2. **Remove sections that have no data** (but include "No information found" notes)
3. **Cite ALL sources** with full URLs in the Sources section
4. **Include timestamps** on all findings where available
5. **Be objective** - present facts, not speculation
6. **Note confidence levels** - distinguish between confirmed facts and possibilities
7. **Respect privacy** - redact unnecessary personal information about innocent parties
8. **Maintain professional tone** - clear, concise, factual
9. **Include disclaimers** - especially about caller ID spoofing and data limitations
10. **Provide context** - help user understand what findings mean
