## Step-by-Step Manual Verification

### Prerequisites

1. **Investigation Phone Setup**
   - iPhone (iOS 14+) or Android (10+)
   - Snapchat app installed and updated
   - Snapchat account registered with Google Voice number
   - Privacy settings configured (see below)

2. **Privacy Configuration**
   - Open Snapchat ‚Üí Profile ‚Üí Settings (gear icon)
   - **Who Can...** settings:
     - Contact Me: My Friends (or Everyone for investigations)
     - View My Story: My Friends
     - See My Location: My Friends (or Ghost Mode)
     - See Me in Quick Add: Off (important!)
   - **Additional Services**:
     - Manage ‚Üí Permissions ‚Üí Contacts: Allow
   - **Mobile Number**:
     - Let Others Find Me: Off (prevents targets from finding you)

### Daily Manual Verification Workflow

#### Step 1: Prepare Investigation Phone

**Add Target Number to Contacts**:

1. Open **Contacts** app on investigation phone
2. Tap **+** (Add Contact)
3. Enter phone number: `+1-555-987-6543`
4. First Name: `Investigation` (or case ID)
5. Last Name: `Target001` (or identifier)
6. **Save**

**Why**: Snapchat syncs contacts to suggest friends. This is a legitimate feature.

#### Step 2: Sync Contacts in Snapchat

**Open Snapchat**:

1. Open Snapchat app
2. Tap **Profile icon** (top left, your Bitmoji)
3. Tap **Add Friends** (+ button)
4. Tap **Contacts** tab

**Grant Access** (if first time):
- "Snapchat Would Like to Access Your Contacts"
- Tap **OK** or **Allow**

**Wait for Sync**:
- Snapchat syncs contacts (takes 5-30 seconds)
- Refresh by pulling down if needed

#### Step 3: Check Results

**If Snapchat Account Exists**:

The contact will appear in one of these locations:

1. **"Contacts" section** (most common)
   - Shows username (e.g., `@johndoe123`)
   - Shows display name
   - Shows Bitmoji (if set)
   - **+ Add** button appears

2. **"Quick Add" section** (if mutual contacts exist)
   - Same information as above
   - May also show mutual friends count

**If No Snapchat Account**:
- Contact does NOT appear in Snapchat
- Only shows up in your phone's Contacts app (not in Snapchat)

#### Step 4: Gather Information

**Tap on the Snapchat profile** (do NOT add them):

**Public Information Visible**:
- ‚úÖ Username (e.g., `@johndoe123`)
- ‚úÖ Display Name (e.g., "John Doe")
- ‚úÖ Bitmoji avatar
- ‚úÖ Snap Score (if public)
- ‚úÖ Public Stories (if they post public Stories)
- ‚ö†Ô∏è **Do NOT add them as friend** (leaves trace)

**Take Screenshots**:
1. Screenshot the profile view
2. Screenshot showing username and display name
3. Save screenshots to secure investigation folder

## Troubleshooting

### Manual Verification Issues

#### Issue: Contact Not Appearing in Snapchat

**Possible Causes**:
1. Number not registered on Snapchat (correct result)
2. Contact sync not enabled
3. Number format incorrect
4. Privacy settings preventing sync

**Solutions**:
```
1. Verify contact sync enabled:
   Snapchat ‚Üí Settings ‚Üí Manage ‚Üí Permissions ‚Üí Contacts: Allow

2. Try different number formats:
   +1-555-123-4567
   15551234567
   (555) 123-4567

3. Force re-sync:
   - Delete contact from phone
   - Re-add contact
   - Snapchat ‚Üí Add Friends ‚Üí Contacts ‚Üí Pull down to refresh

4. Manual verification:
   - Ask the contact directly (if appropriate for investigation)
   - Check publicly posted Snapchat username (if available)
```

#### Issue: Target Appears in "Quick Add" But Not Sure If Correct Person

**Solution**:
- Compare username with other OSINT sources
- Check Bitmoji style (matches personality?)
- View public Story content (if available)
- Cross-reference with WhatsApp/Telegram display names
- Look for mutual friends (could indicate shared network)

#### Issue: Investigation Account Banned

**Cause**: Despite manual verification being safe, account can be banned for:
- Excessive contact sync/unsync patterns
- Too many contact adds/deletes in short time
- Using VPN or suspicious IP addresses
- Other ToS violations

**Solution**:
```
1. Accept the ban (no appeal possible)
2. Get new Google Voice number
3. Register new Snapchat account
4. Wait 24-48 hours before starting checks
5. Reduce check frequency (longer delays between checks)
```

## Best Practices

### For Manual Verification

‚úÖ **DO**:
- Use dedicated investigation phone (not personal)
- Configure privacy settings to minimize trace
- Document all checks with screenshots
- Delete contacts after verification
- Wait reasonable time between checks (hours, not minutes)
- Encrypt screenshots when archiving
- Cross-reference with other platforms (WhatsApp, Telegram)

‚ùå **DO NOT**:
- Add targets as friends (leaves permanent trace)
- Send Snaps or messages to targets
- View their private Stories (they'll see you viewed)
- Sync hundreds of contacts at once (pattern detection)
- Use personal Snapchat accounts

### Rate Limiting (Manual Method)

**Recommended Pace**:
- **5-10 checks per day** maximum
- **Wait 30-60 minutes** between checks
- **Spread throughout day** (not all at once)
- **Delete contacts** after each check

**Why**: Even manual contact syncing can trigger flags if done excessively.

## FAQ

### Q: Why is manual verification recommended for Snapchat but not WhatsApp/Telegram?

**A**: Snapchat has significantly more aggressive anti-automation:
- WhatsApp/Telegram: Moderate ban risk, automation can work for weeks/months with caution
- Snapchat: Extreme ban risk, automation often banned within hours/days

The risk/reward ratio makes manual verification the only practical option for Snapchat.

### Q: Can targets see that I checked them on Snapchat?

**A**: Depends on method:

**Manual Verification**:
- ‚ö†Ô∏è May appear in their "Quick Add" if you have mutual contacts
- ‚ö†Ô∏è Will NOT appear if no mutual contacts (most cases)
- ‚úÖ If you DON'T add them, they won't get notification
- ‚úÖ If you DON'T view their Story, they won't see you

**Automated Verification**:
- ‚ö†Ô∏è May trigger API alerts visible to Snapchat (not user)
- ‚úÖ User won't see anything (if automation works at all)

**Bottom Line**: Manual is low-trace if done carefully.

### Q: How long does manual verification take?

**A**:
- Add contact: 30 seconds
- Sync in Snapchat: 30 seconds
- Check results: 30 seconds
- Take screenshots: 30 seconds
- Document findings: 2 minutes
- Delete contact: 30 seconds

**Total: ~4-5 minutes** per check (including documentation).

For investigations with < 50 numbers, manual is actually **faster** than debugging broken automation.

### Q: What if automated libraries start working again?

**A**: Even if libraries are fixed:
- Ban risk remains extreme
- Snapchat will break them again in next update (within days/weeks)
- Manual method remains more sustainable

**Recommendation**: Stick with manual unless you need to check hundreds/thousands of numbers.

### Q: Can I use Snapchat's search feature instead?

**A**: No.
- Snapchat search only finds usernames, not phone numbers
- Phone number search is blocked by privacy settings
- Contact sync is the ONLY method to verify phone number ‚Üí Snapchat account

### Q: Is there an official Snapchat API I can use?

**A**: No.
- Snapchat offers NO official API for account verification
- Snap Kit API is for app integration, not phone number lookups
- This is intentional (privacy protection)

### Q: What about paying for a OSINT service that includes Snapchat?

**A**: Be cautious.
- Most OSINT services do NOT have official Snapchat access
- They likely use the same unreliable automation (high ban risk)
- May not work reliably
- Check their methodology before purchasing

**Manual verification remains most reliable.**
